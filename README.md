Procedure to Run this website on your local computer.

Installation/Dependencies - 
(1) Install Node.js (NPM included, to access the backend)

To run the program - 
(1) Either Download all the files or clone the git repository using git clone command
(2) Open the backend folder in cmd 
(3) Run command "npm install" when the cmd is in the backend folder (This will install all the required packages for the website namely, Express, body-parser, cors, sqlite3, multer)
(4) Run the command "npm start" when the cmd is in the backend foler (This will generate the databade.db, and create a masteradmin with UserId: admin, Password: pass)
(5) Open the frontend folder, and open index.html
